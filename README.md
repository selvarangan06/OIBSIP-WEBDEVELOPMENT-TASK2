TASK2 => Personal Portfolio Website

## Project Title

Responsive Personal Portfolio Website Using HTML and CSS

## Objective

The primary objective of this project is to design and develop a responsive personal portfolio website that effectively showcases the developer's profile, technical skills, projects, and contact information. The website is designed with a modern and professional user interface to create a positive first impression for recruiters, clients, and visitors. It focuses on providing an attractive layout, smooth navigation, and a responsive design that works seamlessly across desktops, tablets, and mobile devices. The project also demonstrates the practical application of HTML5 and CSS3 concepts, including Flexbox, CSS Grid, media queries, custom properties, and responsive web design principles.

---

## Project Description

A portfolio website serves as a digital resume that highlights an individual's professional profile and technical capabilities. This project is developed as a single-page responsive website containing multiple sections such as Hero, Skills, Projects, Contact, and Footer. Each section is carefully designed to provide essential information in an organized and visually appealing manner.

The Hero section introduces the portfolio owner with a professional headline, short biography, and call-to-action buttons. The Skills section presents technical expertise using interactive cards and progress bars. The Projects section showcases sample works with project descriptions and links for demonstrations. The Contact section provides contact information along with a user-friendly contact form, while the Footer contains social media links and copyright information.

---

## Features

* Fully responsive design for desktop, tablet, and mobile devices.
* Fixed navigation bar with smooth scrolling between sections.
* Professional Hero section with introduction and profile placeholder.
* Skills section displaying proficiency through progress bars.
* Project showcase cards with descriptions and external links.
* Contact section including email, location, LinkedIn, GitHub, and a contact form.
* Modern typography using Google Fonts.
* Clean and elegant color scheme using CSS variables.
* Hover animations and transition effects for better user experience.
* Accessible and semantic HTML structure.

---

Steps Performed

Step 1: Planning the Website Layout

The structure of the portfolio was planned by dividing it into major sections such as Navigation, Hero, Skills, Projects, Contact, and Footer.

 Step 2: Creating the HTML Structure

A semantic HTML5 document was created using elements like <nav>, <section>, <footer>, headings, paragraphs, forms, and links.

Step 3: Styling with CSS

The entire website was styled using CSS3. Custom color variables were defined to maintain a consistent design throughout the website.

Step 4: Navigation Bar

A fixed navigation bar was created with links that allow users to smoothly navigate between different sections of the webpage.

 Step 5: Hero Section

A professional landing page was designed featuring the developer's name, designation, short introduction, profile image placeholder, and action buttons.

 Step 6: Skills Section

Skill cards were developed using CSS Grid. Each card contains an icon, skill name, progress bar, and proficiency level.

Step 7: Projects Section

Project cards were created to display sample projects with titles, categories, descriptions, and links to GitHub repositories or live demos.

 Step 8: Contact Section

A contact area was added containing personal contact details and a responsive contact form with fields for name, email, subject, and message.

 Step 9: Footer

A footer was included with copyright information and social media links.

 Step 10: Responsive Design

Media queries, Flexbox, and CSS Grid were implemented to ensure the website adapts properly to different screen sizes and devices.
